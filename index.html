Zomato UX Case Study

User Experience Analysis & Improvements

1. Overview

Zomato is a popular food delivery and restaurant discovery platform. This case study analyzes its user journey, strengths, weaknesses, and proposes UX improvements.

2. User Journey

  • Open App
  • Search Restaurant or Food
  • Select Item
  • Add to Cart
  • Checkout & Payment
  • Order Tracking

3. Strengths

✔ Clean UI & visual appeal
✔ Smooth checkout process
✔ Real-time order tracking

4. Weaknesses

✖ Too many filters confuse users
✖ Cart charges shown late
✖ Reordering past meals takes effort

5. UX Improvements (with Mockups)

Improvement 1: Simplified Filters

Group filters into Beginner / Advanced sections.

6. Conclusion

Zomato offers a strong UX foundation. With small usability enhancements, the app can further reduce friction and improve user satisfaction.
